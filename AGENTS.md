## AI Agent Workflow

Before implementing any feature:

1. Read SYSTEM_CONTEXT.md
2. Read relevant arc42 architecture docs
3. Read related ADRs
4. Read package CONTEXT.md
5. Create implementation plan
6. Execute
7. Update documentation
8. Commit and push
