# FoodieGo Roadmap

FoodieGo is being developed iteratively based on capability milestones.

## Milestones

- **v0.1 Discovery** ✅
  - Gateway routing
  - Identity Service (Auth)
  - Restaurant Service (Menu, Discovery)
- **v0.2 Shopping** ✅
  - Cart Management (Order Service)
- **v0.3 Checkout** ✅
  - Checkout flow
  - Optimistic concurrency control
- **v0.4 Messaging** ✅
  - RabbitMQ Foundation
  - Outbox Pattern
  - Distributed event publishing
- **v0.5 Observability** ✅
  - OpenTelemetry SDK integration
  - Distributed Tracing (Tempo)
  - Metrics (Prometheus, Grafana)
  - Centralized Logging (Loki)
- **v0.6 Inventory** ⭐ (Next Capability)
  - Reserve stock
  - Confirm/Release reservations
  - Expiration policy
  - Optimistic locking & Exactly-once event handling
- **v0.7 Payment** 🟡 (Pending)
  - Payment gateway integration
- **v0.8 Notification** 🟡 (Pending)
  - Email, SMS, Push via Event Consumption
- **v0.9 Delivery** 🟡 (Pending)
  - Driver matching and location tracking
- **v1.0 Production MVP** 🟡 (Pending)
  - Production readiness and final UI integration
