# 0007. Technology stack selection

## Status
Accepted

## Context
Need to select technologies for the frontend and backend.

## Decision
- **Frontend**: React 18, Vite, TypeScript, Zustand, TailwindCSS, Radix UI.
- **Backend**: Node.js, Express, PostgreSQL, Redis, RabbitMQ.

## Alternatives Considered
- Next.js for Frontend: Rejected as SEO is not the primary concern for the application portals, and SPA offers a simpler deployment model.

## Consequences
- High developer velocity due to familiar stack.
