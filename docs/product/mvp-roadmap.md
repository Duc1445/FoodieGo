# MVP Roadmap

- **Sprint 1**: Foundation (Monorepo, Gateway, Auth).
- **Sprint 2A**: Customer Home, Restaurant Detail, Local Cart.
- **Sprint 2B**: Backend Cart Sync, Checkout Flow.
- **Sprint 2C**: Order Tracking, History, Status Updates.
- **Sprint 3**: Merchant Portal (Menu management, Order receiving).
- **Sprint 4**: Admin Portal (Platform metrics, User management).
