# User Personas

## 1. The Customer (Hungry User)
- **Goal**: Find food quickly, order reliably, and track delivery.
- **Pain Point**: Slow loading menus, failed checkouts, unknown order status.

## 2. The Merchant (Restaurant Owner)
- **Goal**: Manage menu, receive and process orders efficiently.
- **Pain Point**: Missing orders, complex menu management interfaces.

## 3. The Admin
- **Goal**: Oversee platform health, resolve disputes.
