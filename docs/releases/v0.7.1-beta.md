## 8. Technical Debt & Backlog Items

Based on the final CTO review for Sprint 0.7.1, the following Technical Debt items have been recorded and added to the backlog for the upcoming sprint:

1. **[TD-010] Dead Letter Queue Replay & Mock Worker Concurrency:**
   - Implement an admin endpoint or script (e.g., `scripts/replay-dlq.js`) to seamlessly replay events from the Dead Letter Queue.
   - Upgrade the `Mock Gateway Worker` polling strategy to utilize `SELECT ... FOR UPDATE SKIP LOCKED LIMIT 100` instead of a naive un-locked interval tick, allowing the worker to be scaled horizontally across multiple instances safely.

2. **[TD-011] Webhook Signature Key Rotation:**
   - Update `verifySignature` logic in `WebhookController` to support an array of valid secrets (`secret_v1`, `secret_v2`) to allow for zero-downtime key rotation on the Provider/Gateway side.
